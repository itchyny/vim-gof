# vim-gof

![gof](https://user-images.githubusercontent.com/375258/73711173-5c99f600-4749-11ea-9e57-df11a9c31acf.gif)

## Usage
```
:Gof

:let g:gof_mru = 1 " in .vimrc
:Gof mru
```

### Requirements
- Vim with terminal feature (`exists('*term_start')`)
- [gof](https://github.com/mattn/gof)
