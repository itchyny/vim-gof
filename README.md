# vim-gof

![gof](https://user-images.githubusercontent.com/375258/74085564-6448fa00-4abd-11ea-8122-4786b75c9f78.gif)

## Usage
```
:Gof

:let g:gof_mru = 1 " in .vimrc
:Gof mru
```

### Requirements
- Vim with terminal feature (`exists('*term_start')`)
- [gof](https://github.com/mattn/gof)
